## 개요

이전 포스팅에 이어서 개발 과정에서 꼭 필요한 기능과 이슈를 개선하는 내용을 다룹니다.

## NextAuth 커스텀

- NextAuth의 기본 session 타입을 커스터마이징하여 닉네임과 관리자 권한 부여

## React Portal

- React Portal을 사용해 Modal을 구현하여 부모 컴포넌트의 CSS 영향을 받지 않고, 불필요한 렌더링 최소화

## SEO

- 제품 상세페이지는 ISR로 구현하여 SEO 최적화 및 주기적인 데이터 업데이트
- 정적, 동적 Metadata 생성 및 sitemap.xml 파일과 robots.txt를 추가하여 검색 엔진 최적화

## 반응형 디자인

- 다양한 디바이스를 고려한 반응형 디자인 적용
